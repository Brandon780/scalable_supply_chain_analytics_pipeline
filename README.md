# Scalable Supply Chain Analytics Pipeline

## Overview
This project builds an end-to-end data pipeline for retail supply chain analytics using Medallion Architecture (Bronze → Silver → Gold).

## Features
- Sales analytics
- Inventory tracking
- Movement analysis
- Power BI ready dataset
